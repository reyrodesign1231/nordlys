 Nordlys Atlas

Nordlys Atlas is a concept data-platform case study for Arctic expeditions, satellite signals, weather windows, ice conditions, and operational risk monitoring.

The project shows a calm, high-stakes interface for teams that need to read complex environmental data quickly and make decisions with clarity.

## Highlights

- Data-platform concept
- Arctic expedition interface
- Dashboard-style visual system
- Route health and risk monitoring
- Responsive landing page

## Tech Stack

HTML, CSS, JavaScript
